# API com Banco de Dados

## Descrição
API REST com CRUD completo utilizando Node.js, Express e SQLite.

---

## Endpoints

- GET /api/livros
- GET /api/livros/:id
- POST /api/livros
- PUT /api/livros/:id
- DELETE /api/livros/:id

---

## Banco de Dados
SQLite com tabela produtos.

---

## Validações
- Campos obrigatórios no POST
- Tratamento de erro 400, 404 e 500

---

## Testes
Realizados via Postman.

---

## Como rodar

npm install  
node index.js

## Testes com Postman

1. Importe a collection em /postman/Trabalho2-API.postman_collection.json
2. Execute as requisições
3. Base URL: http://localhost:3001
